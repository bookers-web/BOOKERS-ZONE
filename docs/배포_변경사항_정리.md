> 최종 업데이트: 2026-03-03

---

## 1. 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | Bookers Zone (관내 이용자 전용 서비스) |
| Tech Stack | Java 21, Spring Boot 3.4.3, MyBatis 3.0.4, MySQL, JSP |
| 포트 | 8081 |
| 패키지 | `life.bks.zone` |
| 기반 소스 | WWW (프론트), Bookers_Common (공통 로직) → Zone으로 이관 |

---

## 2. 변경 요약

### 핵심 기능
1. **IP 기반 인증 서비스** — 관내 IP로 접속한 이용자 자동 인증 + 동시접속 관리
2. **비회원 도서 열람** — 메인, 주제별, 최신업데이트, 인기, 추천 도서 목록 브라우징
3. **비회원 도서 읽기 (Webviewer)** — Zone 게스트 회원 자동 생성 → DRM 토큰 발급 → 뷰어 리다이렉트
4. **듀얼 DataSource** — Primary(IP 인증 + 레거시 프론트) + Bookers(공통 DB) 분리 운영

## 5. 주요 아키텍처 결정

### 5.1 듀얼 DataSource 구조

```
┌─────────────────────┐     ┌──────────────────────┐
│  Primary DataSource  │     │ Bookers DataSource   │
│  (bookers DB)        │     │ (bookers DB)         │
├─────────────────────┤     ├──────────────────────┤
│ life.bks.zone.mapper │     │ life.bks.zone.mapper │
│ (ipauth, front 등)   │     │         .bookers     │
├─────────────────────┤     ├──────────────────────┤
│ mappers/*.xml        │     │ mappers/bookers/*.xml│
│ mappers/ipauth/*.xml │     │                      │
│ mappers/front/*.xml  │     │                      │
│ mappers/bom/*.xml    │     │                      │
│ mappers/member/*.xml │     │                      │
└─────────────────────┘     └──────────────────────┘
```

- `DataSourceConfig.java`에서 `@MapperScan`의 `excludeFilters`로 `mapper.bookers` 패키지를 Primary에서 제외
- 새 Mapper 추가 시 패키지 위치 주의 필수

### 5.2 Zone 게스트 회원 생성 흐름

```
IP 인증 성공 → PageController (메인 진입)
  → GuestLoginService.findOrCreateZoneGuest()
    → B2bMemberMapper: 기존 회원 조회 (userId = "zone_{uisCode}_{clientIp}")
    → 없으면: MemberMapper: TBL_SEQUENCES에서 다음 번호 조회 (FOR UPDATE)
            → TBL_SEQUENCES 업데이트
            → TBL_MEMBER_MASTER INSERT
            → TBL_B2B_MEMBER INSERT
    → HttpSession에 UM_CODE 저장
```

### 5.3 웹뷰어 연동 흐름

```
바로읽기 클릭 → WebviewerController.getViewerUrl()
  → WebviewerService: 도서 정보 조회 (WebviewerMapper)
  → DrmTokenService: HMAC-SHA256 토큰 발급
  → WVEncryptV2: AES-CBC 파라미터 암호화
  → 뷰어 URL 생성 (devDRM: https://devnewviewer.bookers.life)
```

### 5.4 세션 타임아웃

| 항목 | 값 | 비고 |
|------|-----|------|
| IP 인증 세션 | 5분 | application.yml: `session.timeout-minutes: 5` |
| 쿠키 | 5분 | `ZONE_SESSION` 쿠키 maxAge |

## 10. 주의사항

1. **바로읽기 버튼**: 운영 DB 연결 상태에서 클릭 시 **실제 주문이 생성**됨. 테스트 시 주의.
2. **듀얼 DataSource**: 새 Mapper 추가 시 `mapper/` vs `mapper/bookers/` 패키지 위치에 따라 사용되는 DB가 달라짐.
3. **레거시 코드**: `dao/`, `vo/` 패키지는 WWW에서 이관한 레거시 코드. 새 파일 추가 금지.
4. **DRM 키**: 개발/운영 secret-key가 다름. 배포 환경에 맞게 반드시 변경.
